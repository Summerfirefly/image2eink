# image2eink

This is a small & simple tool to resize and convert PNG/JPEG/BMP image to E-Ink usable BMP image with Floyd-Steinberg algorithm.

Currently the following output image are supported:
- 6-colours (Black, White, Red, Blue, Green, Yellow) with resolution 800x480
